# SpreadsheetCalculator

1) Go to spreadsheetcalculator and compile all exiting java file.
  
  javac src/*.java

2) When all java file compiled run java run to execute Spreadsheet class.
 
 java Spreadsheet

3 2
A2
4 5 *
A1
A1 B2 / 2 +
